Instalación

¿Cómo usar DevOS?

Sigue estos pasos para poner en marcha DevOS en minutos.

Descarga la ISO
Obtén la imagen ISO desde el link de descarga oficial. Tamaño aproximado: 2–3 GB.
Graba en USB o abre en VM
Usa Balena Etcher o Rufus para crear un USB booteable. O simplemente carga la ISO en VirtualBox / QEMU. $ balenaEtcher DevOS-Live_v1.0.iso
Arranca desde USB o VM
Reinicia tu equipo y selecciona el USB en el menú de arranque, o inicia la máquina virtual con la ISO cargada.
Espera y explora
Verás logs durante el arranque. Es normal. Una vez que cargue el escritorio XFCE, puedes comenzar a explorar DevOS en modo Live. Para ver info del sistema instala fastfetch: $ sudo dnf install fastfetch -y
Configura el panel XFCE
Arrastra el panel a la parte inferior, haz clic derecho → Configurar panel. Ajusta el ancho al máximo y bloquéalo.
devos@live ~ $
# Ver versión del sistema
devos@live:~$ cat /etc/os-release
NAME="DevOS"
VERSION="1.0 Live" | ID=fedora
 
# fastfetch no viene preinstalado, hay que instalarlo
devos@live:~$ sudo dnf install fastfetch -y
